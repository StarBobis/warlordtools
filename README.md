## 功能特性

- **过滤器本地编辑**：无需上传网页，快速修改过滤器规则。
- **常用工具集成**：集成了一些常用工具网页

## 开发

```bash
# 第一次下载后安装依赖
bun install

# 本地开发测试
bun tauri dev

# 构建发布版安装包
bun tauri build
```
